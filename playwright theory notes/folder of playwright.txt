playwright.config.js
--------------------
This is the main configuration file for the Playwright project.
It is used to define:
- Browsers (Chromium, Firefox, WebKit)
- Test directory
- Timeouts
- Base URL
- Reporter configuration (HTML, Allure, etc.)
- Headless or headed mode

This file controls how and where Playwright tests run.


package.json
------------
This file contains project metadata and dependencies.
It is used to:
- Define Playwright version
- Install required dependencies
- Manage scripts like:
  - npm test
  - npx playwright test

It acts as the entry point for the Node.js project.


package-lock.json
-----------------
This file locks the exact versions of all installed dependencies.
It ensures:
- Everyone in the team uses the same dependency versions
- No issues due to version mismatch

This file is automatically generated by npm.


.gitignore
----------
This file specifies which files or folders should not be pushed to GitHub.
Commonly ignored files:
- node_modules
- Test reports
- Logs
- Environment files (.env)

It helps keep the repository clean and secure.


tests
-----
This folder contains all Playwright test cases.

Test file naming convention:
filename.spec.js

Example:
login.spec.js
dashboard.spec.js


node_modules
------------
This folder contains all Node.js packages installed for the project.
It includes Playwright and other dependencies.
This folder should not be committed to GitHub.
